C47_DismountVehicle
# Description: robot gets out from the vehicle from a sitting position
facing forward.
Robot gets out with final position standing outside the vehicle in any position.

Known bugs:
* The robot falls out of the car and is not standing stable. 
Thus it can finalize in any posture. But it is out of the vehicle.  

Conditions:
* works with version 1.2 of drcsim!
* start testing world using launch forlder: drc_robot_profile_lower.launch
it is in the launch directory. It also calles a world in the world directory.

*Not final steps, may change*
Follow the steps:
1. roslaunch drc_robot_profile_lower.launch
2. run designer and executer
3. rosrun C47_DismountVehicle C47_DismountV2.py
4. in designer load tree located in TestTree folder.
5. run the tree

Contains:
launch folder: launch files, last working is drc_robot_profile_lower.launch.
world folder: test world, last working world: drc_robot_profile_lower.world.
TestTree folder: tree wich executes the dismounting code.
